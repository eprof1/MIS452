Fall 2010 MIS 452 Management of Telecommunication Systems

Project 2- Computers and Networks

Overview

The purpose of this project is to become familiar with the technology underlying modern computers, computer operating systems, networks and network operating systems.
In order to accomplish this task we will make use of virtualization technology-specifically VMware software.
If you have a PC you will use VMware Workstation.
If you have a Mac you will use VMware Fusion.

You can collaborate with other class members, but each student is responsible for turning in their own project work.

Project 2 Report (1000 points)

Project Steps

These steps will be done gradually during the semester, not all at once.

  1. Make sure hardware assisted virtualization is enabled in your laptop BIOS.
  2. Install virtualization software
    1. PC- Workstation
    2. Mac-Fusion
  3. Complete VMware courses-at a minimum complete the following, other courses are also available
    1. PC-Workstation 7 Fundamentals (90 minutes)
    2. PC-Workstation 7 Advanced (90 minutes)
    3. Mac-VMware Fusion 3 You need only do sections 7 through 11, about 25 minutes
  4. VMware virtual machines needed for this project are found at:
    \\mispgp\VirtualMachineImages
  5. Learn to work with BIOS
    1. note-Boot Order is especially important!
  6. Perform first Operating System (OS) installation-FreeDOS Version 1
  7. Install workstation OS-Windows XP Service Pack 3
  8. Install server OS- Windows Server 2008 (done already in Homework 6)
  9. Submit your report

All work on this project must be finished and uploaded to your web site by 8:00am, Saturday, December 4th, 2010
